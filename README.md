# US Stock Market Analysis

A time-series exploration of US stock market data using Python, rendered as an interactive HTML visualization of price movements over time.

## Overview

This project loads historical US stock market data and explores it as a time series — examining trends, fluctuations, and patterns in prices over time. The output is an interactive time-series chart viewed in the browser as a standalone HTML page (served locally during development).

## Tech stack

- **Language:** Python 3
- **Focus:** time-series analysis and interactive data visualization (HTML output)

## Reproducing the analysis

The interactive chart is generated from a Python time-series workflow built on the standard data stack — pandas and NumPy for data handling, and a plotting library such as Matplotlib, Plotly, or Bokeh for the interactive HTML output. Run the analysis notebook/script, then open the generated HTML file in a browser to explore the chart.

> **Note:** This repository currently tracks the generated visualization shortcut. Committing the source notebook/script alongside it would make the analysis fully reproducible.
